# Portfolio Checklist

Use this checklist to verify your portfolio is complete before submission.

## Data Preparation (40 points)
- [ ] All 5 Sabor Miami data files connected and loaded
- [ ] Null values resolved (removed, replaced, or filled)
- [ ] Inconsistencies fixed (capitalization, spacing, naming)
- [ ] Data types corrected (dates, currency, numbers)
- [ ] Columns renamed with clear, descriptive names

## Data Model (40 points)
- [ ] Star schema visible in Model View
- [ ] Fact table (Sales) at center with dimension tables around it
- [ ] All relationships created with correct cardinality (one-to-many)
- [ ] Date table present and connected to Sales
- [ ] No unnecessary columns loaded into the model

## DAX Measures (40 points)
- [ ] At least 8 working measures created
- [ ] At least 1 CALCULATE measure with a filter
- [ ] At least 1 time intelligence measure (YTD or prior year)
- [ ] Measures displayed in visuals (cards, tables, or charts)
- [ ] Measures use clear, descriptive names

## Independent Dataset (20 points)
- [ ] A dataset of your choice connected to a separate Power BI file
- [ ] At least 3 skills demonstrated (connect + clean + model OR measure)
- [ ] Brief written explanation of what you chose and why

## Reflection (10 points)
- [ ] Written reflection OR recorded screen walkthrough (2-3 minutes)
- [ ] Addresses: What was the hardest part? What would you do differently?

**Total: 150 points**
