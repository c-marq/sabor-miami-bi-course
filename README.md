## About This Course

This is Course 4 of 5 in the Business Intelligence College Certification at Miami Dade College. Students learn to prepare, transform, model, and optimize data in Microsoft Power BI, aligned to the **Microsoft PL-300: Power BI Data Analyst Associate** certification exam.

This repository hosts all instructional materials for the ND (neurodivergent) section, designed using the **Universal Design for Neurodivergence (UDN)** framework.

**Instructor:** Professor Carlos Marquez  
**Term:** Spring 2026  
**Format:** 16 weeks, 2 sessions/week, 100 min/session
